# 🎯 DigiTools

## A-6 Digital Tools Buying Website
A modern, responsive digital tools marketplace rebuilt with React, Tailwind CSS, and DaisyUI. Buy premium digital resources with confidence.

## 🚀 Technology Used
- **React.js** (Frontend UI library)
- **Tailwind CSS & DaisyUI** (Utility-first styling & component library)
- **JavaScript (ES6+)** (Logic & state management)
- **React-Toastify** (Interactive user notifications)

## ✨ 3 Features About The Project
1. **Interactive Cart System:** Seamlessly add products to your cart, check realtime updates on the badge, remove unwanted products, and proceed to checkout with clearing the cart.
2. **Dynamic Product Toggle:** Easily switch between viewing the complete product catalog and managing your curated shopping cart.
3. **Responsive Elegant UI:** A professional and modern user interface that works beautifully across mobile devices, tablets, and desktop computers.

## 🎉 Optional Features Included
- Added badge logic on the Navbar cart icon that updates instantly.
- "Added to cart" message disabled on already bought products on the card.
- Toastify notifications for Add/Remove/Checkout.
